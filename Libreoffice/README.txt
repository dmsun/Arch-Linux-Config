This text describes the process available here - http://www.thinkboxly.com/2015/01/how-to-make-libreoffice-look-awesome.html

1) Install the "Clean Blue" Persona in the Tools - Options - LibreOffice - Personalisation - Own Theme
2) Options - LibreOffice - View -Large Icons
3) Install Custom Icons in "C:\Program Files (x86)\LibreOffice 4\share\config”, or corrsponding dir in Linux
4) Custom Background Colour in Tools - Options - LibreOffice - Application Colours - Application Background Hex Value - #C7D8EE
5) Increase memory limits in Tools - Options - LibreOffice - Memory
6) Disable use anti-aliasing
